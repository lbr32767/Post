# 示例文章 - Markdown 与 MDX 完整指南

欢迎查看这篇综合示例文章！本文将展示 Markdown 和 MDX 的完整用法。

## 第一部分：Markdown 基础

### 文本格式

你可以使用 **粗体**、*斜体*、~~删除线~~ 和 `行内代码` 来强调文本。

### 列表

**无序列表：**

- 第一项
- 第二项
- 第三项

**有序列表：**

1. 第一步
2. 第二步
3. 第三步

### 代码块

**JavaScript 示例：**

```javascript
function hello() {
  console.log('Hello, World!');
}
```

**TypeScript 示例：**

```typescript
interface User {
  name: string;
  email: string;
}

const greet = (user: User): string => {
  return `Hello, ${user.name}!`;
};
```

## 第二部分：MDX 高级特性

### 什么是 MDX？

MDX 是 Markdown 的扩展，允许你在 Markdown 文件中嵌入 JSX 组件。这使得你可以创建更加丰富和交互性强的内容。

### 组件示例

MDX 允许你在 Markdown 中直接使用 JSX 组件，例如：

<div style={{ padding: '1rem', backgroundColor: '#f0f9ff', borderLeft: '4px solid #3b82f6', margin: '1rem 0' }}>
  <strong>提示：</strong>这是一个使用内联样式的示例！MDX 让你可以在 Markdown 中直接使用 JSX。
</div>

### 动态内容

通过 MDX，你可以轻松地在文章中嵌入动态组件，如：

- 交互式图表
- 代码演示
- 自定义提示框

> MDX 结合了 Markdown 的简洁性和 React 组件的强大功能。
